
SDS Portal (EN/AR)

This bundle contains static HTML pages for bilingual Safety Data Sheets (SDS).
To publish online quickly:

Option A — GitHub Pages
1) Create a new public repo on GitHub (e.g., sds-portal).
2) Upload all files to the repo root (index.html at root).
3) Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: /(root).
4) Open the public URL shown under Pages.

Option B — Netlify
1) app.netlify.com → Add new site → Import from Git.
2) Or drag-and-drop the folder/zip to Netlify Drop.
3) Live URL appears after deploy.

Option C — Cloudflare Pages
1) dash.cloudflare.com → Pages → Create a project.
2) Connect Git or upload.
3) Build command: none; Output directory: /
